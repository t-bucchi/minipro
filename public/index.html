An open source program for controlling XGecu's chip programmers.

Minipro is an Open Source tool intended to become a complete cross-platform replacement for the proprietary utility from XGecu. Currently it supports more than 13000 of target devices - including AVRs, PICs as well as a huge number of other microcontrollers and various memory chips.

Features

  • Compatibility with TL866CS, TL866A, TL866II+, T48, and T56 from XGecu (http://www.xgecu.com/en/)
  • More than 13000 target devices (including AVRs, PICs, various BIOSes and EEPROMs)
  • ZIF40 socket and ISP support
  • Vendor-specific MCU configuration bits
  • Chip ID verification
  • Logic IC testing
  • Bitbang support
  • Overcurrent protection
  • System testing
  • Compiles and runs on assorted distributions of Linux, BSD, and MacOS.
